## Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.

## Installing Anon

To install **Venu**, follow these steps:

Linux and macOS:

```bash
sudo git clone https://github.com/venu-3754/Ecommerce_Website.git
```

Windows:

```bash
git clone https://github.com/venu-3754/Ecommerce_Website.git
```
